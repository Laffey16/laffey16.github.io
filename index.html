C# · Unity · LLMs · APIs

Honours Project

My final year project for Abertay asked: can large language models generate enjoyable narrative content in games compared to content written by a human? Built in Unity and C#, the project wraps OpenAI's GPT-3.5 to let players talk to a cast of NPCs around a lake (each with their own prompt and personality) gathering clues to reach the ending.

Whisper handles speech recognition, and Azure TTS handles the voice replies. It was a great hands-on intro to working with LLMs, prompt engineering, and integrating external APIs into Unity, and watching how people actually interact with AI-generated content.

C++ · Emulation · Systems

CHIP-8 Emulator

A working CHIP-8 emulator written from scratch in C++. Implements the full opcode set, display, input, and timers.

Writing this taught me how CPUs actually fetch-decode-execute instructions, how memory maps work, and how to debug at the opcode level.

Built as a learning exercise and a first step towards a Game Boy emulator.
